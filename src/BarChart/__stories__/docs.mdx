## Свойства графика

### Размер элементов

Вы можете менять размер элементов на графике с помощью свойства `size`.
Варианты: `'s'`, `'m'`, `'auto'`.

| `size='s'`               | `size='m`'               |
| ------------------------ | ------------------------ |
| <BarChartExampleSizeS /> | <BarChartExampleSizeM /> |

### Горизонтальный или вертикальный

По умолчанию график горизонтальный. Если вы укажете `isHorizontal={false}`,
график повернется на 90 градусов вправо.

<BarChartExampleNotHorizontal />

### Единицы измерения

За единицы измерения отвечает свойство `unit`.

Вы можете расположить их как удобно, за расположение единиц отвечает `unitPosition`.
Варианты: `left`, `bottom`, `left-and-bottom`, `none`.

| `unitPosition="left"`       | `unitPosition="bottom"`       |
| --------------------------- | ----------------------------- |
| <BarChartExampleUnitLeft /> | <BarChartExampleUnitBottom /> |

| `unitPosition="left-and-bottom"`  | `unitPosition="none"`       |
| --------------------------------- | --------------------------- |
| <BarChartExampleUnitLeftBottom /> | <BarChartExampleUnitNone /> |

## Группы и столбцы

Данные на графике показываются по группам. Это могут быть даты — день, месяц, год, квартал,
или что-то совсем другое, например, города или страны. В любом случае, группа — это параметр,
по которому сравниваются значения.

Например, сколько мл осадков выпало в апреле, мае и июне.

Группы и их содержимое можно описать в объекте `groups`. В него входит:

- название группы — оно будет показываться на графике (задается в `groupName`),
- описание значений для столбцов, которые будут показываться в этой группе (задается в массиве `values`).

Например:

```ts
// groups
[
  {
    groupName: 'первая группа',
    values: [410, 115],
  },
  {
    groupName: 'вторая группа',
    values: [670, 210, 80],
  },
  {
    groupName: 'третья группа',
    values: [75],
  },
]
```

Если название группы не влезает, его можно наклонить. Для этого добавьте `isXAxisLabelsSlanted`
(используйте только в вертикальных графиках).

<BarChartExampleGroupSlanted />

Чтобы спрятать столбцы с пустыми значениями, добавьте `isEmptyColumnsHidden`.

### Цвет столбцов

За цвет столбцов отвечает массив `colors`. Цвета применяются к значениям по порядку — так, как указаны.

```ts
// colors
[
  'var(--color-bg-normal)',
  'var(--color-bg-alert)',
  'var(--color-bg-warning)',
]
```

<BarChartExampleGroup />

### Значения у столбцов

Чтобы показать значения рядом со столбцами, добавьте `showValues={true}`.

<BarChartExampleShowValues />

### Отрицательные значения

Вы можете указать отрицательные и положительные значения, тогда ось будет проходить
по середине графика.

<BarChartExampleGeneral />

```ts
// groups
[
  {
    groupName: 'Q1-2016',
    values: [1000, -100, 50],
  },
  {
    groupName: 'Q2-2016',
    values: [-1000, undefined, 127],
  },
  {
    groupName: 'Q3-2016',
    values: [500, 450, undefined],
  },
  {
    groupName: 'Q4-2016',
    values: [undefined, -300, -200],
  },
  {
    groupName: 'Q5-2016',
    values: [10, 20, 50],
  }
```

## Сетка

### Сколько делений показывать

За количество делений на графике отвечает `gridTicks`.

### Подписи к делениям

Вы можете указать, как часто ставить подписи рядом с делениями: у каждого, у каждого второго, третьего и т. д.
За это отвечает `valuesTicks`.

Формат подписи указывается в `formatValueForLabel`, по умолчанию число — номер деления,
но вы можете показать что-то другое, например, процент или дату.

```tsx
gridTicks={5} //показываем 5 делений на графике
valuesTicks={2} //ставим подпись у каждого второго деления
formatValueForLabel={v => `${v}%`} //добавляем к числу в подписи знак %
```

## Тултип

При наведении на столбец показывается тултип — всплывающее окно с дополнительной информацией об этом столбце.

Вы можете поменять формат значения в тултипе (за это отвечает свойство `formatValueForTooltip`). Например, добавить к значению единицы измерения.

```tsx
formatValueForLabel={v => new Date(v).toLocaleDateString()}
formatValueForTooltip={v => `${v} км`}
```

Так сделано на всех графиках на этой странице: в тултипе к числу добавляется `км` или `%`.

## Предельные значения

Вы можете описать область предельных значений в объекте `threshold`:

- в `value` укажите предельное значение,
- в `color` — цвет, которым его нужно обозначить.

На графике появится пунктирная линия.

<BarChartExampleThreshold />

```ts
// threshold
{
  value: 410,
  color: 'var(--color-bg-normal)',
}
```

## Пример

```tsx
import { BarChart } from '@consta/widgets/BarChart'

<BarChart
  colors={[
    'var(--color-bg-normal)',
    'var(--color-bg-success)',
  ]}
  gridTicks={4}
  groups={[
    {
      groupName: 'Апрель',
      values: [5, 15],
    },
    {
      groupName: 'Май',
      values: [7, 9],
    },
  ]}
  valuesTicks={1}
  size="s"
  threshold={{
    color: 'var(--color-bg-alert)',
    value: 10,
  }}
  isHorizontal
  showValues
  unit="тыс. м³"
  unitPosition="bottom"
  formatValueForLabel={v => `${v} тыс. м³`}
/>
```
