# LegendItem

## Свойства

| Свойство          | Тип                        | По умолчанию | Описание                                                                             |
| ----------------- | -------------------------- | ------------ | ------------------------------------------------------------------------------------ |
| `children`        | `ReactNode`                | -            | Описание рядом с индикатором                                                         |
| `color?`          | `string`                   | -            | Цвет индикатора                                                                      |
| `type?`           | `'dot', 'line', 'warning'` | `'dot'`      | Тип индикатора                                                                       |
| `size?`           | `'xs', 's', 'm'`           | `'s'`        | Размер текста и индикатор                                                            |
| `position?`       | `'top', 'left'`            | `'left'`     | Позиция индикатора                                                                   |
| `isLineBold?`     | `boolean`                  | -            | Если тип индикатора выбран `'line'`, тогда будет применен индикатор утолщенной линии |
| `shouldCropText?` | `boolean`                  | -            | Позволяет включить обрезку строк если их больше чем 2                                |

## Пример

```tsx
import React from 'react'
import { LegendItem } from '@consta/widgets/LegendItem'

const SomeComponent = () => (
  <LegendItem color="red">
    Content
  </LegendItem>
)
```
