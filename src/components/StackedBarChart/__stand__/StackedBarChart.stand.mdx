Столбчатая диаграмма с накоплением: позволяет сравнивать несколько наборов данных друг с другом и одновременно группировать их (например, по времени). Это удобно, если у вас есть несколько типов данных и нужно оценить вклад каждого типа в общий итог.

Похожа на [BarChart](##LIBS.STAND|stand:widgets-components-barchart-stable), но в каждом столбце может быть несколько значений.

> На диаграмме с накоплением не стоит отображать отрицательные значения.
